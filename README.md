## Описание работы URL shorten
``` bash
Короткая ссылка: short.ly/abc123
                 ↓
         GET запрос на сервер
                 ↓
         Сервер берёт "abc123"
                 ↓
           Ищет в Redis:
     Ключ "abc123" → Значение "https://original-long-url.com/very-long-path"
                 ↓
         Находит оригинальный URL
                 ↓
        Отправляет браузеру ответ:
    "HTTP 301 Redirect → https://original-long-url.com/..."
                 ↓
     Браузер автоматически переходит
     на оригинальную длинную ссылку
```
